# Sistema de Categorías

## Cómo funciona

Las categorías en Doña Lina Stock funcionan de manera dinámica y flexible:

### Creación automática
- Las categorías se crean automáticamente cuando agregas un producto con una categoría nueva
- No necesitas crear categorías por separado

### Autocompletado inteligente
- Cuando creas o editas un producto, el campo "Categoría" te sugiere las categorías que ya usaste
- Empieza a escribir y verás sugerencias basadas en tus productos existentes
- Las sugerencias aparecen en orden alfabético

### Crear nuevas categorías
- Si la categoría que necesitas no existe, simplemente escríbela
- Se creará automáticamente cuando guardes el producto
- La próxima vez que crees un producto, esa categoría aparecerá en las sugerencias

## Ejemplos de uso

### Seleccionar una categoría existente
1. Haz clic en el campo "Categoría"
2. Verás una lista desplegable con todas las categorías que ya usaste
3. Selecciona una de la lista

### Crear una categoría nueva
1. Escribe el nombre de la nueva categoría en el campo
2. Guarda el producto
3. La categoría se guardará automáticamente

### Buscar una categoría
1. Empieza a escribir en el campo "Categoría"
2. Las sugerencias se filtrarán automáticamente
3. Ejemplo: escribe "Lác" y verás "Lácteos" si ya existe

## Notas técnicas

- Las categorías se extraen automáticamente de todos los productos existentes
- Se almacenan como parte de cada producto (no hay un archivo separado de categorías)
- Las categorías vacías o duplicadas se filtran automáticamente
- El sistema es case-sensitive: "Lácteos" y "lacteos" se consideran diferentes

## Gestión de categorías

Si quieres cambiar el nombre de una categoría:
1. Edita cada producto que use esa categoría
2. Cambia el nombre de la categoría al nuevo nombre
3. La categoría antigua desaparecerá de las sugerencias cuando no haya productos que la usen
